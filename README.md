# Projectarium

## Sobre o Projeto

O Projectarium é uma plataforma web desenvolvida para o cadastro, organização e acompanhamento de projetos integradores acadêmicos.

O sistema busca funcionar como um acervo digital de projetos, permitindo que trabalhos antigos permaneçam acessíveis para consulta e referência acadêmica, além de auxiliar na organização e comunicação durante o desenvolvimento dos projetos atuais.

---

## Objetivo

O objetivo do Projectarium é centralizar informações relacionadas aos projetos integradores em um único ambiente digital, facilitando:

- cadastro de projetos;
- acompanhamento das equipes;
- armazenamento de documentos;
- comunicação entre alunos e orientadores;
- consulta de projetos antigos.

---

## Funcionalidades Previstas

- Cadastro de usuários;
- Login de alunos e professores;
- Cadastro de projetos integradores;
- Upload de documentos;
- Biblioteca digital de projetos;
- Pesquisa de projetos antigos;
- Controle de status dos projetos;
- Área de comentários e orientações.

---

## Ferramentas Utilizadas

- Trello

## Metodologia

O projeto utiliza conceitos de Scrum e Kanban para organização das tarefas e acompanhamento do desenvolvimento.
